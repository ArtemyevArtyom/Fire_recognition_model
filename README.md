Тема проекта: разработка и обучение модели распознавания возгораний с использованием чат-бота

Цель: разработать модель нейронной сети, интегрированной в приложение чат-бота для распознавания возгораний по фотографии. 

Какую проблему решает: модель может быть использована для детектирования пожаров с камер видеонаблюдения в различных типах помещений, как для частного использования, так и для различных предприятий: в офисах, на складах или на открытой местности, подавая сигналы о возникновении возгораний. Модель будет незаменима в случаях, когда невозможно установить противопожарный автономный датчик или в случае его несрабатывания/ошибочного срабатывания. При фото/видеосъемке и автоматической загрузке изображения в чат-бот, например, с камеры видеонаблюдения, может быть отправлен сигнал в соответствующие службы для быстрого реагирования на возгорание, а также для оповещения людей, находящихся в здании или на территории предполагаемого возгорания.
Детектирование пожара происходит на стыке двух технологий искусственного интеллекта: нейронной сети и чат-бота.
